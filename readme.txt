A CopyBlocker plugin to disable text selection, copying, pasting, and inspecting on your WordPress site.

== Description == CopyBlocker Pro protects your WordPress site by preventing users from selecting, copying, pasting, or inspecting content. With a retro-themed, user-friendly settings interface, you can toggle features like:




Disabling text selection

Blocking copy (Ctrl+C / Cmd+C) and paste (Ctrl+V / Cmd+V)

Preventing select all (Ctrl+A / Cmd+A)

Blocking developer tools/inspect (Ctrl+Shift+I / Cmd+Opt+I)


Disabling right-click context menu

The plugin supports multiple languages, including Bangla, and is designed for ease of use with a modern retro aesthetic.


== Installation ==

Upload the copyblocker-pro folder to the /wp-content/plugins/ directory.


Activate the plugin through the 'Plugins' menu in WordPress.



Navigate to Settings > CopyBlocker Pro to configure restrictions.

== Frequently Asked Questions == = Can I enable restrictions on specific pages? = Currently, the plugin applies to all pages. Contact the author for customizations.

= Is the plugin translatable? = Yes, CopyBlocker Pro is translation-ready and includes support for languages like Bangla (bn_BD).
